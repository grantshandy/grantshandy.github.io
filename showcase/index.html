Wasm4 Playground #

Wasm4 Playground is a website that allows you to write, play, and share WASM-4 retro games. The site works by bundling the AssemblyScript and Roland compilers which compile your source code into portable .wasm files. The site then runs your game in a modified version of the WASM-4 runtime which automatically reloads when you click “Apply Changes”.

Maze Wanderer #

Maze Wanderer is a first-person “walking simulator” for the WASM-4 retro game engine. I wrote this game to learn about raycasting, the method used to create pseudo-3D games before computers had 3D acceleration. The particular algorithm for raycasting took me quite a while to get right, it might be the subject of a future blog post on this site.

Ipmap #

Ipmap is a desktop program that shows the locations of all of the servers that your computer has connected to. It’s fascinating to see that your computer regularly receives information from the other side of the world. Ipmap works by capturing all incoming packets with libpcap, then sending the packet’s source IP address off to an ip-geolocation REST API. Once it gets the response, it simply puts it on the map with some metadata in a context menu.

Unlike other desktop applications with web frontends, Ipmap is a tiny Rust executable that uses your operating system’s preinstalled webview library.

fdg (Force-Directed Graph) #

fdg is a force-directed graph drawing framework for Rust. It’s very primitive in comparison to something like gephi or graphia, but it works well to very quickly visualize small Rust petgraph datasets.

fdg is designed to be as flexible as possible. It supports custom forces as well as rendering graphs to svg, HTML canvas, or interactive windows.

artist-constellations.art #

Artist Constellations is a website that lets you view and shares the artists you listen to on Spotify in a giant graph.

  • Filter by your most listened artists of certain time periods, or by individual genres.
  • View artists by their names, profile pictures, or just simple dots.
  • Share your artists with your friends and see who you have in common and where your tastes intersect.

This site is currently being rewritten in Svelte with a more friendly user interface.

Claui #

Claui is a Rust library that generates convenient GUIs for your clap-based command-line applications. Claui was designed to be as simple and easy to use as possible. In many cases, you can drop it into already-written clap programs and get a high-quality GUI.

// simply declare your program's parameters with clap

claui::run(app, |matches| {
  // then run your application here!
});

// no other configuration is needed!

libretranslate-rs #

LibreTranslate is a web frontend and API for Argos Translate, an open-source translation library. libretranslate-rs is the “official” Rust library for accessing the libretranslate API. libretranslate-rs can translate between English, Arabic, Chinese, French, German, Italian, Japanese, Portuguese, Russian, and Spanish.
